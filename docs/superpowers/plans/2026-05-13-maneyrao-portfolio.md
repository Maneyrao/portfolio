# Maneyrao Studio Portfolio

## Objetivo

Crear un portfolio comercial para Tomi / Maneyrao Studio que venda su perfil como constructor de negocios digitales: ecommerce, landings, sistemas, backend, deploy, WhatsApp checkout y adquisicion con Meta Ads.

## Direccion

- Marca: "Maneyrao Studio — La parte digital de tu negocio, bien armada."
- Tono: directo, operativo, anti-humo, tecnico entendible.
- Diferencial principal: casos con switch "Ver como cliente / Ver como dueno".
- Sensacion visual: editorial-operativa, estudio boutique + consola de negocio + comercio real.

## Secciones

1. Hero con posicionamiento, prueba rapida y CTA a WhatsApp.
2. Tira de capacidades y evidencia.
3. Casos en produccion con lectura cliente/dueno.
4. Servicios agrupados por vender, operar, escalar y adquirir.
5. Proceso de trabajo.
6. Perfil de Tomi sin pitch.
7. Contacto con formulario que arma mensaje a WhatsApp.

## Verificacion

- Tests unitarios para datos, vistas de caso y URL de WhatsApp.
- Lint.
- Build de produccion.
- Revision visual local en navegador.
